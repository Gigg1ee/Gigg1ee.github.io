思考题

  1. Linux and Linux?

    Ubuntu和debian都非常适合做服务器操作系统。

    Ubuntu有亮丽的用户界面，完善的包管理系统，强大的软件源支持，丰富的技术社区，并且Ubuntu对计算机硬件的支持好于debian，兼容性强，应用多。

    debian整个系统基础核心非常小，与Ubuntu相比，没有那么花哨，底层稳定，占用硬盘空间和内存小，同时，debian的帮助文档和技术资料比较少。

  2. Can’t memory be larger?

    内存不能无限大，最大支持的内存是由操作系统和硬件决定。

    对于32位操作系统，限制为4GB。在计算机中，CPU的地址总线数目，决定了CPU的寻址范围。（一般32位的CPU的地址总线为32位）那么32位提供的可寻址的物理地址范围为2^32 = 4GB。

  3. Why Windows is quite ‘fat’?

    因为Windows能够兼容各种各样不同的硬件和很多旧的版本的Windows，所以Windows系统中会有很多很多的驱动程序，以及各种旧版本的API实现，占的内存就多。

  4. Why executing the ‘poweroﬀ’ command requires superuser privilege?

    防止普通用户在不知情的情况下关闭计算机，导致数据损失，所以需要root权限。

  5. What happened?

    make命令使得nemu目录下的文件每一个进行“预处理 -> 编译 ->汇编 -> 链接”生成可执行目标程序。

  6. How will you do?

    通过数电的学习，可以设计出简单的加法器、乘法器、寄存器、计数器这些可以组合起来组建成计算机的硬件。

    包括算术逻辑单元和处理器寄存器的运算器、包括指令寄存器和程序计数器的控制器、存储器。

    还需要设计一套合适的指令集和微指令，把各种门电路结合到一起，最后接上输入输出设备。

实验内容

0.1 Install a GNU/LINUX VM

  1. 安装Virtual Box与Debian虚拟机

    打开Oracle VM VirtualBox管理器；

    点击新建，建立一个新的虚拟机，设置名称ics，类型选择linux，版本选择Debian(32-bit)。

    设置内存：1024MB

    设置硬盘：选择现在创建虚拟硬盘;

    • 硬盘文件类型选择VDI；
    • 存储在物理硬盘上默认选择；
    • 文件位置和大小默认路径保存，大小设置8GB

  2. 在虚拟机中安装操作系统

    1）加载Debian安装映像

    • 选中新建的虚拟机ics，点击工具栏的设置，点击存储

      然后选中没有盘片，在右侧属性中点击蓝色的光盘，选择Choose/Create a Virtual Optical Disk...

      随后选择在网上下载的镜像文件debain-10.3.0-i386-netinst.iso，最后点击OK。

      设置完毕即可点击启动启动虚拟机

2）安装debian

  • 用键盘的上下键选择Graphical Install，按Enter进入，同时，断开右下角的互联网连接；

  • 选择语言，English - English

  • 选择位置，other -> Asia -> China

  • 配置语言环境，保留默认设置United States - en_US.UTF-8

  • 配置键盘，American English

  • 等待CD加载安装程序组件

  • 配置网络

    • 网络自动配置失败。

      不管警告，点击Continue

    • 选择Do not configure the network at this time然后Continue

    • 设置主机名：保留默认值debian

  • 设置用户和密码

    • 根密码：设置成root

  • 新用户名的全名：设置成zhangwanping

  • 设置用户密码：zwp123

  • 配置时钟：此步骤将被跳过，等待加载完毕。

  • 分区磁盘

    • 选择Guided - use entire disk。
    • 选择列表中唯一的磁盘。
    • 分区方案选择All files in one ....
    • 查看分区信息无误后，选择Finish partitioning and write changes to disk，
    • 选择yes。

  • 安装基本系统中，等待加载。

    • 选择No。

  • 配置程序包管理器：Continue without a network mirror? -选择Yes。

  • 参加包装使用情况调查？选择No（Yes也可以）

  • 软件选择：确保选择了standard system utilities，确定Continue，等待加载。

  • 将GRUB引导加载程序安装到硬盘上

    • 选择Yes。
    • 选择/dev/sda。

  • 完成安装
